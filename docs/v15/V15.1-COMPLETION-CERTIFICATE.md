## 2. Mission

Turn TODAY from a dashboard into an operating system. The operator opens TODAY and
immediately knows what to do — they complete actions, they do not consume information.

This is a **presentation-only** restructure. The V15 brain is unchanged; what changed is
what the operator *sees*.

---

## 3. What changed (TODAY surface)

| Mission | Change |
|---|---|
| 1 — Front door | TODAY keeps the nav grid (open/close/stock-count/counter/insights/playbooks/guide/setup) as a secondary "More" zone, so the operator rarely needs to leave. |
| 2 — Dashboard retirement | **Removed** the "How the shop is doing" status panel entirely — it changed no behaviour ("the shop is doing well" is not an action). |
| 3 — Do Now zone | New dominant, bordered, raised `do-now-zone` — the first and largest thing on the page. ≤3 numbered (1·2·3) action cards. The day-shape + guided walk fold in as a slim lead-in so nothing competes. |
| 4 — Later zone | Collapsed by default, muted, secondary; sits below Do Now. |
| 5 — Action focus | Each card answers **What** (title) → **Why** (one line: money at stake when priceable, else what's happening) → **do it** (the card opens the action). Nothing else. |
| 6 — Stress test | Validated by journey: all priorities sit **above the fold** (no scrolling) so they read in one glance. |
| 7 — Information demotion | Weekly summary demoted to a collapsed, muted "For reference" panel **below** Do Now — it can never outrank actions. |
| 8 — Visual hierarchy | Do Now: 2px green border + shadow, 2xl heading, numbered cards. Everything else: muted backgrounds, smaller headings, collapsed. |

No brain/logic changes: `buildOwnerBrain`, the compression engine, confidence routing and
V14 truth are untouched.
